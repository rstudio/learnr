# learnr Tutorial Examples

## Complete Tutorials

These tutorials were designed to showcase the features of the learnr
package — and to teach learners how to use R! Try working through these
tutorials to get a sense of the look and feel of a learnr tutorial. Then
check out the source code to see how these tutorials were made!

[![Preview image of Setting Up
R](images/tutorial-ex-setup-r.png)](https://learnr-examples.shinyapps.io/ex-setup-r/)

##### [Setting Up R](https://learnr-examples.shinyapps.io/ex-setup-r/)

A tutorial featuring videos and interactive questions to guide a new R
user through the installation and set up of everything they'll need to
get started with R.

[Source](https://github.com/rstudio/learnr/tree/main/inst/tutorials/ex-setup-r/ex-setup-r.Rmd)

[![Preview image of Filtering
Observations](images/tutorial-ex-data-filter.png)](https://learnr-examples.shinyapps.io/ex-data-filter/)

##### [Filtering Observations](https://learnr-examples.shinyapps.io/ex-data-filter/)

An example tutorial teaching a common `data` transformation: *filtering*
rows of a data frame with `dplyr::filter()`.

[Source](https://github.com/rstudio/learnr/tree/main/inst/tutorials/ex-data-filter/ex-data-filter.Rmd)

[![Preview image of Summarizing
Data](images/tutorial-ex-data-summarise.png)](https://learnr-examples.shinyapps.io/ex-data-summarise)

##### [Summarizing Data](https://learnr-examples.shinyapps.io/ex-data-summarise)

An example tutorial where learners are introduced to
`dplyr::summarise()`. Along the way, learners also gain practice with
the pipe operator, `%>%`, and `dplyr::group_by()`.

[Source](https://github.com/rstudio/learnr/tree/main/inst/tutorials/ex-data-summarise/ex-data-manip-summarise.Rmd)

## Demo Tutorials

There’s a lot that learnr can do. These tutorials showcase specific
learnr features or show you some possibilities that build on other
packages designed to work with R Markdown and learnr.

[![Preview image of learnr: Quiz
Questions](images/tutorial-quiz_question.png)](https://learnr-examples.shinyapps.io/quiz_question/)

##### [learnr: Quiz Questions](https://learnr-examples.shinyapps.io/quiz_question/)

A brief introduction to the types of interactive questions you can use
in a learnr tutorial with the
[`learnr::question()`](https:/pkgs.rstudio.com/learnr/v0.11.6/reference/quiz.md)
and
[`learnr::quiz()`](https:/pkgs.rstudio.com/learnr/v0.11.6/reference/quiz.md)
functions.

[Run Tutorial](https://learnr-examples.shinyapps.io/quiz_question/)

[![Preview image of learnr: slidy
layout](images/tutorial-slidy.png)](https://learnr-examples.shinyapps.io/slidy/)

##### [learnr: slidy layout](https://learnr-examples.shinyapps.io/slidy/)

learnr tutorials can even be used in slides! This example tutorial is a
short presentation that showcases an interactive exercise that produces
interactive visualizations with
[htmlwidgets](http://www.htmlwidgets.org/).

[Source](https://github.com/rstudio/learnr/tree/main/inst/tutorials/slidy/slidy.Rmd)

[![Preview image of learnr: SQL
Exercises](images/tutorial-sql.png)](https://learnr-examples.shinyapps.io/sql-exercise/)

##### [learnr: SQL Exercises](https://learnr-examples.shinyapps.io/sql-exercise/)

You can use learnr to teach students how to work with databases using
interactive SQL exercises. In this example, we describe how to create
your own SQL exercises using the gradethis package to check the
student's query.

[Source](https://github.com/rstudio/learnr/tree/main/inst/tutorials/sql-exercise/sql-exercise.Rmd)

[![Preview image of sortable: Ranking
Questions](images/tutorial-external-sortable.png)](https://andrie-de-vries.shinyapps.io/sortable_tutorial_question_rank/)

##### [sortable: Ranking Questions](https://andrie-de-vries.shinyapps.io/sortable_tutorial_question_rank/)

The [sortable](https://rstudio.github.io/sortable/) package allows you
to create custom HTML widgets with drag-and-drop capabilities that you
can even use in learnr tutorials. This tutorial shows you what's
possible with sortable and how to add drag-and-drop questions to your
tutorials.

[Source](https://github.com/rstudio/sortable/tree/main/inst/tutorials/question_rank/question_rank.Rmd)

## Community Showcase

learnr’s community has built some excellent tutorials and resources for
learning data science and programming. Here are just a few highlights.
(Please [suggest your favorite learnr
resources](https://github.com/rstudio/learnr/discussions/categories/show-and-tell).)

[![Preview image of
DoSStoolkit](images/example-community-dosstoolkit.png)](https://rohanalexander.github.io/DoSStoolkit/)

##### [DoSStoolkit](https://rohanalexander.github.io/DoSStoolkit/)

The [Department of Statistical Sciences](http://www.utstat.utoronto.ca/)
(DoSS) toolkit is a free series of open source online modules written by
undergraduates, that their fellow students and the public can use to
learn the essentials of R.

[GitHub](https://github.com/RohanAlexander/DoSStoolkit/)

[![Preview image of
learntidymodels](images/example-community-learntidymodels.png)](https://github.com/tidymodels/learntidymodels)

##### [learntidymodels](https://github.com/tidymodels/learntidymodels)

Learn [tidymodels](https://www.tidymodels.org/) in the browser or
locally in your RStudio IDE!

[GitHub](https://github.com/tidymodels/learntidymodels)

[![Preview image of
qsslearnr](images/example-community-qsslearnr.png)](https://github.com/mattblackwell/qsslearnr)

##### [qsslearnr](https://github.com/mattblackwell/qsslearnr)

This package contains learnr tutorials based on [*Quantitative Social
Science: An Introduction*](http://qss.princeton.press/) by [Kosuke
Imai](https://imai.fas.harvard.edu/) from Princeton University Press.

[GitHub](https://github.com/mattblackwell/qsslearnr)

[![Preview image of Between the
Pipes](https://github.com/meghall06/personal-website/raw/8de36739/static/img/pipes_logo.png)](https://github.com/meghall06/betweenthepipes)

##### [Between the Pipes](https://github.com/meghall06/betweenthepipes)

`betweenthepipes` is an R package with two tutorials, useful for
learning tidyverse basics with hockey data.

[GitHub](https://github.com/meghall06/betweenthepipes)

[![Preview image of
afrilearnr](images/example-community-afrilearnr.png)](https://afrimapr.github.io/afrilearnr/)

##### [afrilearnr](https://afrimapr.github.io/afrilearnr/)

A growing collection of interactive learnr tutorials for learning
spatial data analytics while making African maps.

[GitHub](https://github.com/afrimapr/afrilearnr)

[![Preview image of
RTLedu](images/example-community-rtledu.png)](https://github.com/risktoollib/RTLedu)

##### [RTLedu](https://github.com/risktoollib/RTLedu)

[RTLedu](https://github.com/risktoollib/RTLedu) contains learnr
tutorials to accompany the RTL R package by
[@risktoollib](https://github.com/risktoollib).

[GitHub](https://github.com/risktoollib/RTLedu)
